🇮🇳 KAVIZ Tamil — Phonetic Encoding Rules (T1, T2, T3)
Rule T1 — Consonant-Digit Unitary Logic (உயிர்மெய் ஒருங்கிணைப்பு விதி)
Every Tamil consonant = ONE fixed digit. All vowel-combined forms (உயிர்மெய்) = SAME digit. Vowel does NOT change digit!
த — தௌ = 4  |  ம — மௌ = 6  |  ழி, ழா, ழு = all digit 8
Example: தமிழன் → த(4) + ம(6) + ழ(8) + ன்(NULL) = 468
Rule T2 — Null-Value Letters (மதிப்பற்ற எழுத்துக்கள் விதி)
Three categories carry NO digit value (NULL):
① மெய்யெழுத்து (18 pure consonants with pulli): க் ங் ச் ஞ் ட் ண் த் ந் ப் ம் ய் ர் ல் வ் ழ் ள் ற் ன் = NULL
② ஆய்த எழுத்து: ஃ = NULL
③ Grantha compound letters: ஸ்ரீ, க்ஷ = NULL (in any form)
Example: கப்பல் → க(1) + ப்(NULL) + ப(5) + ல்(NULL) = 15
Example: லக்ஷ்மி → ல(9) + க்ஷ(NULL) + மி(6) = 96
Rule T3 — Phonetic Prosody Duration Mapping (ஒலி அளவு விதி)
Digit 0: குறில் உயிரெழுத்துக்கள் (அ,இ,உ,எ,ஒ) — ண வரிசை / ன வரிசை = 0
ண வரிசை: ண,ணா,ணி,ணீ,ணு,ணூ,ணெ,ணே,ணை,ணொ,ணோ,ணௌ = 0
ன வரிசை: ன,னா,னி,னீ,னு,னூ,னெ,னே,னை,னொ,னோ,னௌ = 0
குறில் உயிர் தனியாக வந்தாலும் = digit 0!

Digit 9: நெடில் உயிரெழுத்துக்கள் (ஆ,ஈ,ஊ,ஏ,ஓ,ஐ,ஔ) — ல வரிசை / ள வரிசை = 9
ல வரிசை: ல,லா,லி,லீ,லு,லூ,லெ,லே,லை,லொ,லோ,லௌ = 9
ள வரிசை: ள,ளா,ளி,ளீ,ளு,ளூ,ளெ,ளே,ளை,ளொ,ளோ,ளௌ = 9
நெடில் உயிர் தனியாக வந்தாலும் = digit 9!

Example: அரசன் → அ(0) + ர(7) + ச(2) + ன்(NULL) = 072
Example: மாலை → ம(6) + லை(9) = 69
Example: ஆலமரம் → ஆ(9) + ல(9) + ம(6) + ர(7) + ம்(NULL) = 9967
🇬🇧 KAVIZ English — Phonetic Encoding Rules (E1, E2, E3, E4)
Rule E1 — Diagonal Cross Vowel Rule
Before splitting — check each vowel:
① If vowel NOT touching any consonant → diagonal cross (/) it → skip!
② Split remaining letters by 2 from left to right

BEAUTIFUL: B E A U T I F U L → BE+UT+IF+UL = 0538
YOU ARE TRUE: Y O U A R E T R U E → YO+AR+ET+RU = 8353
YOU ARE IN QUEUE: Y O U A R E I N Q U EUE → YO+AR+IN+QU = 8302
Rule E2 — Double Consonant Priority Rule
When two consonants appear together → first consonant gives digit → second = NULL (absorbed into pairing)!
TH = T(5) + H(NULL) = 5
DR = D(2) + R(NULL) = 2
KN = K(7) + N(NULL) = 7
SS = S(4) + S(NULL) = 4
Example: MISS YOU = MI(9)+SS(4)+YO(8) = 948
Example: FRAUD = FR(3)+UAD(2) = 32
Rule E3 — Reversal Rule
Order doesn't matter — consonant ALWAYS gives digit! Vowel+Consonant = reverse = Consonant+Vowel.
AD → DA → D = 2
ER → RE → R = 3
IL → LI → L = 8
Example: HEAD = HE(5)+AD(2) = 52
Example: I LOVE YOU = IL(8)+OV(6)+EY(8) = 868
Rule E4 — Last Consonant Rule
Last consonant of word/sentence = keeps its digit alone! Any vowel after last consonant = diagonal cross → skip!
Example: KAVIZ = KA(7)+VI(6)+Z(9) = 769 → Z alone ✅
Example: SEVEN = SE(4)+VE(6)+N(0) = 460 → N alone ✅
Example: COME = CO(1)+M(9)+E = 19 → E after M = skip! ✅
📝 Sentence Formation (User Method)
Long numbers split into groups freely — ANY combination! 1+1, 1+2, 2+3, 2+4, 4+2 — whatever is comfortable!
Each group = one word. All words = one memorable sentence!
Example: 699189 = JUMP(69)+MUCH(91)+LAZY(89) = "JUMP MUCH LAZY"
Example: Gandhi(69)+Ambedkar(91)+Nehru(89) = 699189 🇮🇳
⚡ Y is always a Consonant in KAVIZ!
Y is NOT a vowel in KAVIZ English. Y = digit 8 (along with L).
Example: YAMAHA = YA(8)+MA(9)+HA(5) = 895
Example: YOU = YO(8) — Y gives digit!

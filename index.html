📖 KAVIZ Encoding Rules
📌 Core Principle: Entire sentence = ONE continuous stream. Spaces ignored. Split strictly LEFT→RIGHT by 2 letters. CONSONANTS give digits. VOWELS are helpers only.
Rule E1 — Diagonal Cross Vowel Rule (Core Rule)
Split every 2 letters left to right.
• Vowel + Consonant → Consonant gives digit (E3 Reversal)
• Consonant + Vowel → Consonant gives digit
• Vowel + Vowel → Skip 1 vowel (Diagonal Cross ✗) — NO digit

Examples:
KAVIZ = KA(7) + VI(6) + Z(9) = 769 ✅
I LOVE YOU = IL(5) + OV(9) + EY(0) = 590 ✅
YOU ARE TRUE = YO(8) + AR(3) + ET(5) + RU(3) = 8353 ✅
BEAUTIFUL = BE(0) + UT(5) + IF(3) + UL(8) = 0538 ✅
Rule E2 — Double Consonant Priority Rule
Two consecutive consonants → FIRST gives digit, SECOND = NULL.

Common pairs: TH=5 | DR=2 | KN=7 | SS=4 | CH=1 | SH=4 | WH=7

Example:
MISS YOU = MI(M=9) + SS(S=4, second S=NULL) + YO(Y=8) + U(✗) = 948 ✅
I MISS YOU = IM(9) + IS(4) + SY(4) + OU(✗) = 944 ✅
Rule E3 — Reversal Rule
Order does NOT matter — CONSONANT always gives digit regardless of position.
Vowel+Consonant = Consonant+Vowel = same digit.

Examples:
AD = DA = D = 2 | ER = RE = R = 3
IL = LI = L = 8 | OV = VO = V = 6
HEAD = HE(H=5) + AD(D=2, reversal) = 52 ✅
Rule E4 — Last Consonant Rule
LAST consonant of sentence keeps its digit alone — no vowel pair needed.
Any vowel AFTER last consonant = Diagonal Cross ✗ = skip.

Examples:
KAVIZ = ...Z(9, E4) = 769 ✅
SEVEN = SE(4) + VE(6) + N(0) = 460 ✅
THESE = TH(5,E2) + ES(4) + E(✗) = 54 ✅
Special — Y is Always Consonant (Digit 8)
Y is NEVER a vowel in KAVIZ — always treated as consonant digit 8.

Examples:
YA=8 | YO=8 | EY=8 | AY=8 | YAMAHA = YA(8)+MA(9)+HA(5) = 895 ✅

📌 முக்கியக் கொள்கை: உயிர்மெய்யில் மெய் பகுதி மட்டும் digit தரும். மெய்யெழுத்து தனியாக வந்தால் NULL. உயிரெழுத்து T3 விதிப்படி digit தரும்.
Rule T1 — உயிர்மெய் ஒருங்கிணைப்பு விதி (Consonant-Digit Unitary Logic)
ஒவ்வொரு மெய் எழுத்தும் ஒரு fixed digit தரும்.
உயிர்மெய்யில் உயிர் மாறினாலும் digit மாறாது — மெய் பகுதி மட்டும் digit தரும்.

எடுத்துக்காட்டு:
க, கா, கி, கீ, கு, கூ, கெ, கே, கை, கொ, கோ, கௌ = அனைத்தும் 1
ம, மா, மி, மீ... = அனைத்தும் 6
த, தா, தி, தீ... = அனைத்தும் 4

தமிழ் = த(4) + மி(6) + ழ்(NULL) = 46 ✅
கப்பல் = க(1) + ப்(NULL) + ப(5) + ல்(NULL) = 15 ✅
Rule T2 — மதிப்பற்ற எழுத்துக்கள் விதி (Null-Value Letters)
மூன்று வகை எழுத்துக்கள் digit தராது (NULL):

① ஆய்த எழுத்து (ஃ) = எப்போதும் NULL
  எ.கா: எஃகு = எ(0) + ஃ(NULL) + கு(1) = 01 ✅

② மெய்யெழுத்து (18 புள்ளி எழுத்துக்கள்) = எப்போதும் NULL
  க் ங் ச் ஞ் ட் ண் த் ந் ப் ம் ய் ர் ல் வ் ழ் ள் ற் ன் = NULL
  எ.கா: கப்பல் → ப்(NULL), ல்(NULL)

③ கிரந்த கூட்டெழுத்துக்கள் (ஸ்ரீ, க்ஷ) = எந்த வடிவிலும் NULL
  ஆனால்: ஜ, ஷ, ஸ, ஹ = ACTIVE = Digit 5
  எ.கா: ஹாய் = ஹா(5) + ய்(NULL) = 5 ✅
Rule T3 — ஒலி அளவு விதி (Phonetic Prosody Duration Mapping)
உயிரெழுத்துக்கள் இரண்டு வகை digit தரும்:

Digit 0 — குறில் உயிர் (Short Vowels):
அ, இ, உ, எ, ஒ = 0
ண வரிசை (ண,ணா,ணி...) = 0
ன வரிசை (ன,னா,னி...) = 0

Digit 9 — நெடில் உயிர் (Long Vowels):
ஆ, ஈ, ஊ, ஏ, ஓ, ஐ, ஔ = 9
ல வரிசை (ல,லா,லி...) = 9
ள வரிசை (ள,ளா,ளி...) = 9

எடுத்துக்காட்டு:
சென்னை = ச(2) + ன்(NULL) + ன(0) + ய்(NULL) = 20 ✅
தமிழ் நாடு = த(4) + ம(6) + ழ்(NULL) + ந(3) + ட(3) = 4633 ✅
